This repository contains the source code and static website files for my personal E-portfolio. The website showcases my background, education, professional experience, projects, technical skills, certifications, awards, and personal interests.

## About This Portfolio

This E-portfolio was created to present my personal brand as a data-driven business professional with interests in marketing strategy, business analytics, project management, and business innovation.

The website includes the following pages:

- **Home**: Personal introduction, professional focus, featured projects, and quick highlights
- **About**: Education, career direction, and personal interests
- **Experience**: Work experience, leadership experience, awards, and professional development
- **Projects**: Selected academic and professional projects
- **Skills**: Technical skills, certifications, and language abilities
- **Contact**: Contact information and professional links
